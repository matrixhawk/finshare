## 二、实施计划

### 阶段一：核心数据 (第1-2周)

| 任务 | 数据源 | API |
|------|--------|-----|
| 期货K线 | 大商所/郑商所 | 仓里财富 |
| 期货实时 | 上海期货 | 官方接口 |
| 基金净值 | 天天基金 | 官方API |

**里程碑：期货数据对齐 akshare**

### 阶段二：市场扩展 (第3-4周)

| 任务 | 数据源 | API |
|------|--------|-----|
| 港股实时 | 腾讯/新浪 | 实时API |
| 可转债 | 东方财富 | 数据中心 |
| 指数数据 | 中证指数公司 | 官方API |

**里程碑：覆盖 A股 + 港股 + 期货 + 基金**

### 阶段三：深度数据 (第5-6周)

| 任务 | 数据源 | API |
|------|--------|-----|
| 美股数据 | Yahoo Finance | 免费API |
| 基金详情 | 天天基金 | 持仓/规模 |
| 新股数据 | 东方财富 | 打新日历 |
| 龙虎榜详情 | 东方财富 | 营业部明细 |

**里程碑：数据深度对齐 akshare**

### 阶段四：特色功能 (第7-8周)

| 任务 | 描述 |
|------|------|
| 因子库 | 技术因子、基本面因子 |
| 舆情分析 | 公告、新闻 |
| 筹码分布 | 股东户数、持仓成本 |
| 实时监控 | WebSocket 推送 |

**里程碑：差异化超越**

## 五、API 设计规范

### 5.1 命名规范 (对齐 akshare + 简化)

```python
# akshare 风格 (保留兼容)
ak.stock_zh_a_spot_em()           # A股实时
ak.fund_etf_hist_em()             # ETF历史
ak.futures_zh_daily_sina()        # 期货日线

# finshare 风格 (更简洁)
fs.get_snapshot('000001.SZ')      # 实时
fs.get_fund_nav('161039')          # 基金净值
fs.get_future_kline('IF2409')     # 期货K线
```

### 5.2 统一返回格式
```python
# 所有接口统一返回 DataFrame
df = fs.get_fund_nav('161039')
print(df)
#     code  name    nav   nav_acc  change  change_pct  date
# 0  161039 易方达  1.234  1.456  0.012      0.98%  2024-01-02
```
